## Overview

This repository contains Python implementations for the ITSFA4-14 Time Series Analysis and Forecasting Project 1. The project analyses and forecasts the popularity of three in-demand tech job roles — **Data Analyst**, **Software Engineer**, and **Cybersecurity Specialist** — based on weekly Google search interest scores from April 2019 to April 2024.

The dataset (`tech_roles_trends.csv`) is used as a proxy for career awareness, learning demand, and hiring relevance, supporting LinkedIn's labour market insights mission.

## Model Chosen: SARIMA

**SARIMA(p,d,q)(P,D,Q)[52]** was selected for all three job roles.

Key justification:
- Weekly data with strong **annual seasonality** (s=52) confirmed via decomposition
- **Non-stationarity** (trend) addressed by first-order differencing (d=1, D=1)
- **ACF/PACF** patterns after differencing guide p and q selection
- **AIC grid search** ensures objective, data-driven parameter selection per role
- **Ljung-Box** residual tests confirm white noise residuals → model adequacy
